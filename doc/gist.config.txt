*gist.config.txt*  CreateGist configuration

DESCRIPTION
    The `:CreateGist` command can be configured to avoid prompting the user for the privacy settings of the Gist and target clipboard.
    This is done via the `setup()` function with the `clipboard` and `private`.

OPTIONS
	private = false, will set the privacy to `public` by default.
	clipboard = "0", will set the clipboard to `+` by default, it is the unnamed_plus register `:h quoteplus`.
